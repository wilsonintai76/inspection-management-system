# **App Name**: Inspectable

## Core Features:

- Inspection Scheduling: Create and manage inspection schedules, assigning locations, supervisors, and auditors.
- Department Management: Add and manage departments to categorize inspection locations.
- Location Management: Define and manage inspection locations, linking them to departments and assigning default auditors.  Allows for specification of two auditors.
- Auditor Management: Register and manage a list of available auditors for assignment to inspection schedules.
- User Role Management: Manage user roles (Auditor, Supervisor, Admin) to control access and permissions within the application.
- Inspection Analysis Tool: Generative AI tool that recommends changes based on an inspection based on its auditor feedback. Tool uses current weather data, current news, recent news of equipment recalls to decide on specific feedback it should incorporate.
- User Profile Management: Allow users to manage their profiles.

## Style Guidelines:

- Primary color: Emerald green (#50C878) to represent safety, health, and growth.
- Background color: Light-green (#E0F8E6) for a clean and calming backdrop.
- Accent color: Teal (#008080) for interactive elements and calls to action.
- Headline font: 'Space Grotesk', sans-serif, for headlines and short amounts of body text; if longer text is anticipated, use this for headlines and 'Inter' for body
- Body font: 'Inter', sans-serif, used for the main body text.
- Use consistent and clear icons from Lucide Icons library to represent actions and data types.
- Use a card-based layout for presenting information, with clear sections and spacing for readability.
